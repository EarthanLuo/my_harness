## Agent skills

### Issue tracker

Issues live as markdown files under `.scratch/<feature>/` in this repo. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles, each using its default string. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.

## Harness bundle

`.agents/skills/`, `.codex/hooks*`, and `.claude/` are generated by `harness/generate.js` from `third-party/` submodules per `harness/manifest.json` (rationale in `docs/harness/skills-reconciliation.md`). To change skill or hook selection, edit the manifest and re-run the generator. Do not hand-edit generated bundles because they are overwritten; local modifications go in `harness/overlays/`.
