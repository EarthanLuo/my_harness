## Agent skills

### Issue tracker

Issues live as markdown files under `.scratch/<feature>/` in this repo. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles, each using its default string. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.

## Harness bundle

`.claude/skills/` is generated by `harness/generate.js` from `third-party/` submodules per `harness/manifest.json` (rationale in `docs/harness/skills-reconciliation.md`); to change skill selection, edit the manifest and re-run the generator; do not hand-edit `.claude/skills/` (it is overwritten) — local modifications go in `harness/overlays/`.
